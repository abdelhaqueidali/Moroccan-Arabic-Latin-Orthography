# Moroccan-Arabic-Latin-Orthography
An Amazigh-Inspired Latin Orthography for Moroccan Arabic

A standardized Latin-script orthography for Moroccan Arabic (Darija), modeled after the regional Amazigh-Latin system. It will feature writing rules, annotated text examples, song lyrics, and mini-dictionary glossaries. This is just to showcase the usage of this orthography, which could benifit others who are looking for such solutions.
